# Minimal Example

This folder contains a minimal, practical setup for using `opencode-postmortem-plugin`.

Files:

- `opencode.json` - minimal plugin configuration for a project
- `commands.md` - command-by-command usage guide with examples

How to use this example:

1. Copy `opencode.json` to your project root (or merge the plugin entry into your existing config).
2. Run `npx postmortem-init` to install command and skill templates.
3. Restart OpenCode.
4. Use `commands.md` as your quick reference while working incidents.

For full installation details, see [INSTALL.md](../../INSTALL.md).
